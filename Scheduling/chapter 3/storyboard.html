Figure 17

Scheduler Homepage

The user will be directly going to the homepage of the scheduler system after the user successfully login to the system. The homepage consists of buttons which are the main functional requirements of the system. Those buttons are the modules of the system that has its specific submodule.

Figure 18

Manage Schedule

This page is where the user can manage the course-schedule. The user can click the “Add Schedule” button if it wants to add course-schedule record. They will be going to another page if the add schedule button is being clicked by the user and they are needed to fill-up the form so that they can add another record to the database. If the user wants to search for a specific record, the user needs to input the “Schedule ID”. They can also click the department search box if they want to look for a schedule that is under in a specific department. And if the user wants to edit/update specific record, they will be needed to click edit button besides the specific record and they will be going to another page to fill up the form for editing. In the other hand, if the user wants to delete specific record they will be going to click the delete button on the right side.
